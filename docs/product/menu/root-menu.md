# Root Menu

## Goal

Top-level menu with navigation to all subsystems.

## Layout

- 54 slots (6 rows).
- Row 2 (centered):
  - Slot 11: `Skills` (diamond pickaxe)
  - Slot 12: `Stats` (book)
  - Slot 13: `Leaderboard` (gold ingot)
  - Slot 14: `Settings` (redstone)
  - Slot 15: `Help` (oak sign)
- Row 6:
  - Slot 49: `Close` (barrier)

## Rules

1. Clicking a navigation item opens the target menu.
2. Clicking empty slots does nothing.
3. Closing the inventory returns to normal gameplay.
4. Menu title and item names use the player's selected language.
5. The Rules button has been removed; Rules now live inside Help.
