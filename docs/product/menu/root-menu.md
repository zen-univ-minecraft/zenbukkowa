# Root Menu

## Goal

Top-level menu with navigation to all subsystems.

## Layout

- 27 slots (3 rows).
- Row 1:
  - Slot 10: `Skills` (diamond pickaxe) -> opens Skills menu
  - Slot 12: `Stats` (book) -> opens Stats menu
  - Slot 14: `Leaderboard` (gold ingot) -> opens leaderboard page
  - Slot 16: `Settings` (redstone repeater) -> opens Settings menu
- Row 2: empty.
- Row 3: empty.

## Rules

1. Clicking a navigation item opens the target menu.
2. Clicking empty slots does nothing.
3. Closing the inventory returns to normal gameplay.
4. Menu title: `zenbukkowa`.
