# DISCOVERY Skills

### CURIOUS_MINER
- **Mechanic:** Unlocks the DISCOVERY point category.
- **Tier 1:** First-break bonuses enabled.
- **Prerequisite:** AREA_RADIUS tier 1.

### GEOLOGIST
- **Mechanic:** Increases common block discovery bonuses.
- **Tier 1:** +5 points.
- **Tier 2:** +10 points.
- **Tier 3:** +15 points.
- **Prerequisite:** CURIOUS_MINER.

### SURVEYOR
- **Mechanic:** Increases uncommon block discovery bonuses.
- **Tier 1:** +10 points.
- **Tier 2:** +20 points.
- **Tier 3:** +30 points.
- **Prerequisite:** GEOLOGIST.

### CARTOGRAPHER
- **Mechanic:** Increases rare block discovery bonuses.
- **Tier 1:** +15 points.
- **Tier 2:** +30 points.
- **Tier 3:** +45 points.
- **Prerequisite:** SURVEYOR.

### PATHFINDER
- **Mechanic:** Increases epic block discovery bonuses.
- **Tier 1:** +50 points.
- **Tier 2:** +100 points.
- **Tier 3:** +150 points.
- **Prerequisite:** CARTOGRAPHER.

### WORLD_WALKER
- **Mechanic:** Multiplies all discovery bonuses.
- **Tier 1:** 1.1x multiplier.
- **Tier 2:** 1.2x multiplier.
- **Tier 3:** 1.3x multiplier.
- **Prerequisite:** PATHFINDER.
