# Skill Tree

## Goal

List every skill, its category, tiers, and exact effect.

## TERRA Skills

### AREA_RADIUS

- Tiers: `1`, `3`, `5`, `7`, `9`
- Effect: horizontal break radius in blocks from center
- Tier 1 = 1x1 (radius 0), tier 5 = 9x9 (radius 4)

### AREA_DEPTH

- Tiers: `1`, `3`, `5`, `7`, `9`
- Effect: vertical break depth in blocks from target Y
- Tier 1 = 1 layer, tier 5 = 9 layers

## MINERAL Skills

### HASTE_AURA

- Tiers: `I`, `II`, `III`, `IV`, `V`
- Effect: permanent haste level equal to tier number

### FORTUNE_TOUCH

- Tiers: `I`, `II`, `III`
- Effect: `tier * 10%` chance to grant double points on any break

## ORGANIC Skills

### LEAF_CONSUME

- Tiers: `off`, `on`
- Effect: leaves are included in area breaks without requiring shears or tool

### ROOT_RAZE

- Tiers: `off`, `on`
- Effect: when breaking a log, also break logs directly below in the same column

## AQUATIC Skills

### SALVAGE

- Tiers: `I`, `II`, `III`
- Effect: `tier * 15%` chance to not consume tool durability on area breaks

### TIDE_BREAKER

- Tiers: `off`, `on`
- Effect: water breathing and aqua affinity while underwater

## VOID Skills

### VOID_SIPHON

- Tiers: `I`, `II`, `III`
- Effect: `tier * 25%` bonus points for blocks at Y < 0 or in Nether/End dimensions

### STRUCTURE_SENSE

- Tiers: `off`, `on`
- Effect: highlights nearby structure blocks with particles when holding a tool
