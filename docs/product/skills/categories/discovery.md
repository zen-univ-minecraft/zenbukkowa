# DISCOVERY — Exploration and First Breaks

## Identity

DISCOVERY rewards exploring new terrain and breaking block types for the first time. It is the curiosity category.

## Typical Blocks

Any naturally occurring block type broken for the first time.

Full list: [point-categories.md](../../point-categories.md#discovery)

## Skill Theme

- **Unlock:** `CURIOUS_MINER` enables DISCOVERY points.
- **Rarity boost:** `GEOLOGIST`, `SURVEYOR`, `CARTOGRAPHER`, and `PATHFINDER` increase bonuses for common, uncommon, rare, and epic blocks respectively.
- **Multiplier:** `WORLD_WALKER` multiplies all discovery bonuses.

## Strategy Note

The DISCOVERY branch is unique: it does not depend on volume of breaking, but on variety. Players who explore many biomes earn the most. Unlock `CURIOUS_MINER` early to start accumulating one-time bonuses.
