# Localization

## Goal

Support multiple languages for in-game text and documentation.

## Supported Languages

- English (`en`) — default
- Japanese (`ja`)

## In-Game

- Players switch language via Settings menu.
- Preference is saved to SQLite and persists across sessions.
- All menu titles, item names, lore, chat messages, and help text are localized.

## Files

- `src/main/resources/lang/en.yml` — English strings
- `src/main/resources/lang/ja.yml` — Japanese strings
