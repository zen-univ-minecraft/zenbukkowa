# ゲームルール

## 概要

zenbukkowa は時間制限付きのブロック破壊イベントです。プレイヤーはブロックを破壊してカテゴリポイントを獲得し、そのポイントを消費してスキルを解放し、破壊力を高めます。

## スコアリング

- 5つのカテゴリでポイントを獲得: TERRA、MINERAL、ORGANIC、AQUATIC、VOID。
- 基本ブロック = 1ポイント。鉱石 = 5ポイント。古代の残骸 = 20ポイント。
- 構造物（海底神殿、森の洋館）を破壊すると +5000 ポイント。

## スキル

- 5つのカテゴリにわたる26のスキルをカテゴリポイントで解放します。
- スキルにはティアと前提条件があります。最も強力な能力を解放するためのパスを構築しましょう。
- 最大範囲破壊 = 1回の破壊で9x9x9ブロック。

## イベントの流れ

1. オペレーターが `/zenbukkowa start` または設定メニューからイベントを開始します。
2. スコアボードに経過時間とスコアが表示されます。
3. オペレーターが `/zenbukkowa end` または設定メニューからイベントを終了します。
4. 総ポイントに基づいて勝者が発表されます。

## 制限事項

- スキルの購入は返金できません。
- コンテナ（チェスト、シュルカーなど）は範囲破壊から保護されています。
- 各構造物ボーナスはサーバーごとに1回のみ獲得可能です。
