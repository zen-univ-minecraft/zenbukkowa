# Gameplay Rules

## Goal

Explain the event rules to players in plain language.

## Rules

1. Break blocks to earn points in six categories: TERRA, MINERAL, ORGANIC, AQUATIC, VOID, CROP.
2. Spend points to unlock skills that expand break area and speed.
3. First tier of any skill costs 50 points.
4. Destroy structures for huge bonuses.
5. Highest total score wins.
6. No refunds on skill purchases.
7. Containers and bedrock are protected.
