# Gameplay Rules

## Goal

Explain the event rules to players in plain language.

## Rules

1. Break blocks to earn points in six categories: TERRA, MINERAL, ORGANIC, AQUATIC, VOID, CROP.
2. Spend points to unlock skills that expand break area and speed.
3. Every domain has its own area-expansion skill — balance all six to maximize destruction.
4. First tier of any normal skill costs 50 points.
5. Mythic skills at the top of the tree cost multiple category types at once.
6. Destroy structures for huge bonuses.
7. Highest total score wins.
8. No refunds on skill purchases.
9. Containers and bedrock are protected.
