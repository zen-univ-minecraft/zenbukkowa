# Game Rules

## Overview

zenbukkowa is a timed block-breaking event. Players earn category points by breaking blocks, then spend those points to unlock skills that increase destruction power.

## Scoring

- Break blocks to earn points in five categories: TERRA, MINERAL, ORGANIC, AQUATIC, VOID.
- Base blocks = 1 point. Ores = 5 points. Ancient debris = 20 points.
- Destroy structures (Ocean Monument, Woodland Mansion) for +5000 points.

## Skills

- Spend category points to unlock skills across 5 categories.
- First tier of every skill costs 50 points of its category.
- Skills have tiers and prerequisites. Build a path to unlock the most powerful abilities.
- Max area destruction = 9x9x9 blocks per break.
- See [skill-descriptions.md](../skills/skill-descriptions.md) for all skill details.

## Event Flow

1. An operator starts the event with `/zenbukkowa start` or from the Settings menu.
2. The scoreboard shows elapsed time and scores.
3. An operator ends the event with `/zenbukkowa end` or from the Settings menu.
4. The winner is announced based on total points.

## Restrictions

- No refunds on skill purchases.
- Containers (chests, shulkers, etc.) are protected from area breaking.
- Each structure bonus can only be claimed once per server.
- Operators can reset player data with `/zenbukkowa reset`.
