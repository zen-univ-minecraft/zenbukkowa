# Gameplay Rules

## Goal

Explain the event rules to players in plain language.

## Rules

1. Break blocks to earn points in seven categories: TERRA, MINERAL, ORGANIC, AQUATIC, VOID, CROP, and DISCOVERY.
2. Spend points to unlock skills that expand break area and speed.
3. Every domain has its own area-expansion skill -- balance all seven to maximize destruction.
4. First tier of any normal skill costs 50 points.
5. Mythic skills at the top of the tree cost multiple category types at once.
6. Destroy structures for huge bonuses.
7. Discover new block types for one-time DISCOVERY bonuses.
8. Highest total score wins.
9. No refunds on skill purchases.
10. Containers and bedrock are protected.
11. Player-placed blocks do not award points.
12. Immature crops do not award points.
