# Purpose

## Goal

Deliver a 2-hour event plugin where breaking blocks generates multi-category points, spending points unlocks skills that expand break area and speed, and the highest cumulative score wins.

## Outcomes

- Players feel constant progression through skill unlocks.
- Area destruction creates satisfying feedback loops.
- Structure bonuses reward exploration and target selection.
- All point operations happen through the Nether Star menu.
- Scoreboard keeps competition visible at all times.

## Non-Goals

- Backward compatibility with older Minecraft versions.
- Multi-server network sync.
- Persistent economy beyond the event session.
- Human-first narrative docs over machine-retrieval clarity.
